# core-user-service

## Installation

### macOS / Linux (bash/zsh)

```bash
python3.13 -m venv .venv
source .venv/bin/activate
pip install -U pip
pip install -e .

```

### Windows(PowerShell)

```bash
py -3.13 -m venv .venv
.\\.venv\\Scripts\\Activate.ps1
python -m pip install -U pip
pip install -e .
```

### Windows(CMD)

```bash
py -3.13 -m venv .venv
.\.venv\Scripts\activate.bat
python -m pip install -U pip
pip install -e .
```
